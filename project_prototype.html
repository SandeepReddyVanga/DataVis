Dataset Summary

  • Fertilizer imports/exports data - USA
  • Data is available for 26 major fertilizers and 82 trading countries
  • Data is available in both volume(tons) as well as dollar ammount
  • Imports data is available from 1995 to 2012
  • Exports data is available from 1990 to 2012
More details about the dataset can be found here.

Sketch Summary

I plan to visualize different aspects of data set using
  • Multi series line chart: Time series of US imports/exports total volume/dollar ammount for each fertilizer
  • Choropleth & Circle packing: Country wise annual US imports/exports
  • Scatter-Plot Matrix & Heat Map: Co-occurences of fertilizer imports/exports over time
