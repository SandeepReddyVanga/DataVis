C-level Functionality

  • Query string is obtained from page URL, decoded and converted to lower case
  • Spaces, punctuations and special characters are ignored and letter counts are obtianed
  • Letters and corresponding counts are logged to the console
  • Basic Bar chart is of these counts is constructed

Customization

  1. Number of Y-Axis tick labels is changed to 3
  2. Size of inner and outer tick labes of Y-Axis differs
  3. Number of decimal points displayed on Y-Axis is changed
  4. Vowels are displayed in steel blue color and consonants are displayed in light blue
  5. Letters are sorted by alphabet to begin with

B-level Functionality

  • Two buttons are added. One is "Sort by Value" and the other is "Sort by Alphabet". The functionality is obvious from the name of the button.

A-level Functionality

Discussion

This visualization displays count of alphabets obtained from query string. In case the query string obtained from URL is empty, The default query string is "Default Letters". I chose to display Vowels in steel blue and consonants in light blue to differentiate between them. I also reduced the number of ticks down to 3 as the exact count value is displayed when mouse is hovered on the bar. I added two seperate buttons to sort by value or alphabet.
